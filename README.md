# DIY CAR-T At Home

### Steps
1. Isolate T cells from blood.
2. Genetically engineer T cells to express chimeric antigen receptor (CAR).
3. Proliferate CAR T cells ex vivo.
4. Infuse back into patient. 

### Antigens
1. CD19
2. CD20
3. CD22
4. CD30 (TNFRSF8) 
5. CD33
6. CD123 (IL3R)
7. CD135 (FLT3)
8. BCMA
